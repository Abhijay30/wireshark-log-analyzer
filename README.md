# Wireshark Log Analyzer

A Python-based PCAP investigation tool built using PyShark and TShark.

## Features

* Host Discovery
* Protocol Analysis
* Top Talkers Analysis

## Technologies

* Python
* PyShark
* Wireshark
* TShark

## Usage

```bash
python analyzer.py
```
